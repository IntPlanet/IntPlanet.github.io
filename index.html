    公式显示

    在Markdown文件头部添加MathJax网页版脚本可显示公式（参见：Getting Started with MathJax Components），需要添加三段JavaScript脚本：

    // 浏览器兼容
<script src="https://polyfill.io/v3/polyfill.min.js?features=es6"></script>

// 行内公式标识兼容性修改
<script>
MathJax = {
  tex: {
    inlineMath: [['$', '$'], ['\\(', '\\)']]
  }
};

// MathJax解析引擎
</script>
<script type="text/javascript" id="MathJax-script" async
  src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js">
</script>

    公式测试

    行内公式

    这是行内公式：\(a^2+b^2=c^2\)，\(\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}\)。行内公式最好不用$...$实现，而是用\\(...\\)实现，从而可以有效避免了美元符号$潜在的不正常显示问题。

    行间公式

    这是行间公式：

    $$ a^2+b^2=c^2 $$

    整点复杂的：

    $$ \sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6} $$
