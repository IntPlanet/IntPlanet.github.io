    MkDocs：快速、简单且华丽的静态站点生成器

    MkDocs 简介

    MkDocs 是一款基于 Python 语言编写的专注于创建项目文档的静态站点生成器，它快速、简单且界面华丽。

    MkDocs 使用 Markdown 来撰写文档源码, 并用 YAML 文件作为配置文件管理整个站点的内容。当创建新的项目后，MkDocs 会自动生成一个配置文件 mkdocs.yml 和一个 docs 文件夹。 docs 文件夹用于存放 Markdown 文件， mkdocs.yml 作为则用于配置这些文件在最终站点页面的排布方式。

    MkDocs 自带两款主题，分别为 mkdocs 和 readthedocs，其中 mkdocs 为默认主题。当然，一个流行的站点生成器少不了优秀的第三方主题（参见：mkdocs.themes），本站使用的主题为 Material for MkDocs。

    安装 MkDocs

    确保操作系统具有 Python 环境和 pip 工具后，采用如下命令安装MkDocs：

    pip install mkdocs

    安装完成后，可以检查以下 MkDocs 是否安装成功：

    mkdocs --version
>>> mkdocs, version 1.0.4 from c:\anaconda3\lib\site-packages\mkdocs (Python 3.7)

    若能看到输出版本号且无任何报错，即表示 MkDocs 安装成功。

    MkDocs 命令

    MkDocs 通过命令来生成项目及站点文件，常用命令如下：

    • mkdocs new [dir-name] - 创建新项目；
    • mkdocs serve - 启动本地服务并实时更新站点文件修改；
    • mkdocs build - 构建静态站点文件；
    • mkdocs gh-deploy - 将站点文件部署到Github仓库；
    • mkdocs help - 打印帮助信息。

    MkDocs 项目结构

    mkdocs.yml    # 站点配置文件
docs/
    index.md  # 站点主页
    ...       # 其他 Markdown 页面，图片及其他文件

    页面公式显示

    MathJax 网页脚本

    在 Markdown 文件头部添加 MathJax 网页版脚本可显示公式（参见：Getting Started with MathJax Components），需要添加三段 JavaScript 脚本：

    // 浏览器兼容性设置
<script src="https://polyfill.io/v3/polyfill.min.js?features=es6"></script>

// 行内公式标识修改
<script>
MathJax = {
  tex: {
    inlineMath: [['$', '$'], ['\\(', '\\)']]
  }
};

// MathJax解析引擎设置
</script>
<script type="text/javascript" id="MathJax-script" async
  src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js">
</script>

    公式测试

    行内公式

    这是行内公式：\(a^2+b^2=c^2\)。行内公式最好不用 $...$ 实现，而是用 \\(...\\) 实现，从而可以有效避免了美元符号 $ 潜在的不正常显示问题。

    行间公式

    这是行间公式：

    $$ a^2+b^2=c^2 $$

    整点复杂的：

    $$ \sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6} $$

    MkDocs 站点部署

    MkDocs 使用 mkdcos build 命令直接构建完整的静态 HTML 站点文件（位于 site 文件夹内），这些文件无需修改即可部署到服务器。

    MkDocs 站点文件部署到 Github 等代码托管站也非常简单，使用命令 mkdocs gh-deploy ，MkDocs 就会自动将相应内容推送到项目的 gh-pages 分支上，然后只需要在项目设置中选择好对应的 Github Pages 分支，稍等片刻即可通过 https://user-name.github.io/ 或 https://user-name.github.io/project-name/ 访问。
