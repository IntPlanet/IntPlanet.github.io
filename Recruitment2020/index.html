    深圳市魔方卫星科技有限公司简介¶

    深圳市魔方卫星科技有限公司，以打造人人都用得起得卫星为使命，专业从事微小卫星核心产品的全平台、高品质、批量化研制。公司致力于高性价比商业卫星的核心单机产品研发与系统设计，致力于打造高性价比的遥感卫星系统，最终实现卫星数据的消费级商业化应用。

    核心成员均具备近10年的航天行业从业经验，负责和管理过十多颗卫星的论证、制造、管理和运营。熟悉航天行业的技术准则、行业标准和设计生产制造流程，同时具备卫星工程（结构力学、姿轨控、星务计算机、热控、数传载荷、总体电路），星座建设，卫星测控，卫星应用服务等专业的全产业链技术人员，具有丰富的设计生产经验。

    魔方卫星的单机产品定位为高品质、低成本、快响应和模块化，在保证产品高可靠性要求的前提下，大幅度的降低成本，缩短生产周期，满足商业微小卫星系统的市场需求。魔方卫星不仅具备全平台核心单机产品研制能力，还具备整星的设计、生产、运营能力。通过打造高性价比的微小卫星系统，带动卫星应用市场的发展，最终实现卫星数据的消费级商业化应用，构建以应用为主导的新航天商业模式，切实服务于广大中小企业和消费级用户。

    卫星系统应用部门 2020 年热招岗位¶

    遥感图像处理算法工程师（1人）¶

    岗位职责

    • 负责遥感图像产品需求调研、应用分析与用户挖掘；
    • 负责遥感图像处理算法研究，遥感应用产品的工程化生产；
    • 负责遥感图像产品项目，进行数据统计分析，形成专题图、报告；
    • 研究遥感图像的前沿算法，如图像增强、智能分类、大尺度时序分析等。

    任职要求

    • 卫星遥感、空间摄影测量、GIS 等相关等专业硕士及以上学历，自学能力强、善于表达与沟通、乐于主动解决问题；
    • 精通遥感数据处理与加工流程，熟练使用 ENVI、ArcGIS 等专业图像分析软件，熟悉 OTB、OpenCV 等开源图像处理工具并具备独自进行二次开发能力者优先；
    • 熟练使用 Python 或 JavaScript 编程语言；
    • 熟悉深度学习的基础理论和主流算法、了解目前流行的基于深度学习的图像优化算法，并掌握以下一种深度学习框架者优先：TensorFlow、PyTorch、Caffe等。

    薪资待遇

    • 15-25k/月（14薪），能力突出者可面议。

    WebGIS 开发工程师（1~2人）¶

    岗位职责

    • 负责基于 CesiumJS 的 WebGIS 产品开发与项目实施工作；
    • 负责系统模块的编码、测试、部署及说明文档编制；
    • 负责 GIS 产品系统架构设计、系统模块分析和系统接口设计；
    • 配合完成其他各类技术开发任务。

    任职要求

    • GIS 或计算机相关专业，硕士及以上学历，自学能力强、善于表达与沟通、乐于主动解决问题；
    • 熟练使用 JavaScript 编程语言及 CesiumJS 框架，从事过 WebGIS 项目开发；
    • 熟悉 OpenLayers、Leaflet、Mapbox、Skyline 等至少其中一种流行 WebGIS 框架，有1年或以上的 WebGIS 系统运维经验者优先；
    • 熟悉 Electron 框架、ArcGIS Sever 或开源 WebGIS 服务器（如 GeoServer、MapServer、MapProxy 等）者优先；
    • 熟悉 SpatialLite、PostgreSQL、MySQL 等开源数据库者优先。

    薪资待遇

    • 15-25k/月（14薪），能力突出者可面议。

    GIS 应用工程师（1~2人）¶

    岗位职责

    • 负责遥感影像服务的发布、测试与管理；
    • 负责 GIS 数据服务建设、应用开发和接口开发；
    • 负责 GIS 应用软件的设计、实现、测试及维护等工作；
    • 参与客户应用需求分析与技术支持、系统维护。

    任职要求

    • 卫星遥感、GIS 或计算机等相关等专业硕士及以上学历，有较强的语言表达、沟通能力和分析、解决问题的能力；
    • 了解测绘、地理信息相关基础知识，精通GIS基础理论知识，熟练掌握至少一款 GIS 专业软件，如 ArcGIS、QGIS、PostGIS 等，并有实际开发经验，能够独自进行二次开发；
    • 熟练使用 Python 或 JavaScript 编程语言；
    • 熟悉 WebGIS 系统和服务开发，有三维 GIS 开发经验者优先
    • 熟悉 SpatialLite、PostgreSQL、MySQL 等开源数据库者优先。

    薪资待遇

    • 15-25k/月（14薪），能力突出者可面议。
