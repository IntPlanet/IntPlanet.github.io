    Google Earth Enterprise (OpenGEE)¶

    Earth Enterprise 是谷歌地球企业版（Google Earth Enterprise，GEE）的开源实现，简称 OpenGEE，谷歌于2017年3月将其开源并托管在 Github: google/earthenterprise。

    GEE 是一款地理空间应用程序，可用于构建和托管自定义 3D 地球和 2D 地图。OpenGEE 不提供 Google 地图或 Google 地球中当前可用的 Google 影像及私有图库。

    OpenGEE 包含三大核心组件：

    • Fusion：将图像，矢量和地形数据源导入并融合到单个可漫游的 3D 地球或 2D 地图中，已完全开源。
    • Server：基于 Apache 或 Tornado 的服务器，托管由 Fusion 构建的私有地球模型，已完全开源。
    • Client：用于查看 3D 地球模型和 2D 地图的 Google 地球企业版客户端（Earth Client，EC）和 Google Maps 的 Javascript API V3。其中 EC 免费下载和使用，Google Maps 的 API Javascript API V3 已完全开源。
